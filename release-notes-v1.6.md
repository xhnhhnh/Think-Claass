# 🚀 Think-Class v1.6.0 正式发布：教与学的“题库-练习-纠错”闭环上线

本次 **v1.6.0** 带来了一套更贴近真实教学流程的能力升级：从教师出题到学生练习，再到错题沉淀与巩固复习，形成完整的学习闭环，并配套更清晰的数据结构与接口层实现，便于后续扩展与运维。

## ✨ 版本亮点

- **试卷系统（Teacher/Student）**
  - 教师端新增试卷编辑与管理能力（出题、维护、发布）
  - 学生端新增试卷列表与作答流程
- **错题本（Wrong Questions）**
  - 学生练习沉淀错题，便于针对性复习与二次练习
- **学习计划（Study Plans）**
  - 支持按学习节奏组织内容，提升可持续学习体验
- **知识体系（Knowledge / Knowledge Graph）**
  - 新增知识点与知识体系相关能力，为后续“知识图谱 + 个性化推荐”打基础

## 🧩 技术与结构调整

- **后端**
  - 新增多组业务路由：试卷、作答提交、学习计划、错题本、知识点等
  - 数据结构同步扩展（Prisma Schema 更新）
- **前端**
  - 新增对应 API 封装与 Query Hooks，页面按角色落地（Teacher/Student）
  - 相关布局与全局状态做了必要的适配，确保模块可用与可扩展

## 📦 部署与更新

- 本版本仍提供生产部署包（zip），配合一键安装/更新脚本可快速上线
- 如果你使用 GitHub Release 部署，请下载对应版本附件：
  - `think-class-v1.6.0.zip`
