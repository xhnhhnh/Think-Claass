## 3. W3C HTML5 / CSS3 规范校验

- **DOCTYPE 声明**: 使用标准的 `<!DOCTYPE html>`。
- **Meta 标签**: 包含了 `charset="UTF-8"`、`viewport` 以及 SEO 相关的 `description` 和 `keywords`。
- **语义化标签**: 广泛使用了 `<nav>`, `<header>`, `<main>`, `<section>`, `<footer>` 等 HTML5 语义化标签。
- **冗余与废弃标签**: 无使用 `<font>`, `<center>` 等废弃标签。
- **校验结果**: 通过 W3C Validator 基础校验，无结构性错误，所有标签均正确闭合。

## 4. 链接与资源检查

- **内部链接 (无死链)**: `index.html`, `about.html`, `services.html`, `news.html`, `contact.html` 之间的互相跳转全部测试通过。
- **相对路径引用**: CSS、JS 脚本的引用均采用严格的相对路径 (`./css/style.css`, `./js/main.js`)。

## 5. 功能交互验证

- **导航栏高亮**: JS 脚本 `main.js` 能根据当前页面的 URL，自动为对应的导航项添加 `active` 类。
- **轮播图组件**: 自动轮播与手动控制 (前后按钮、指示器) 正常。
- **联系表单验证**: HTML5 表单自带验证 (`required`, `type="email"`) 与 Bootstrap 样式结合正常；JS 成功拦截默认提交事件，避免页面刷新，并能动态弹出成功提示框。
